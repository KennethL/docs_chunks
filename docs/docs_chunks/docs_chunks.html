    edoc_to_chunk/1

    (since 0.0.1) 
    edoc_to_chunk(string()) -> docs_v1().

    Fetch edoc docs from a given ErlPath and convert it to docs chunk.

    http://erlang.org/eeps/eep-0048.html

    Examples:

      %% docs_chunks:edoc_to_chunk("src/foo.erl").
  #=> {docs_v1, ..., erlang, <<"text/markdown">>", ..., ..., ..., ...}
